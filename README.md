A bilingual practice test for the California DMV written exam, built to serve Farsi-speaking communities who lacked a study tool in their own language. Questions are drawn from a verified bank grounded in the official California Driver's Handbook, with each question citing its source, and presented in English and Farsi/Dari. Built with Next.js, TypeScript, and Tailwind CSS; uses browser localStorage to track seen questions and avoid repeats within a session. Deployed on Vercel
